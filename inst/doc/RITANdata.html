Introduction to RITANdata

Michael T. Zimmermann

2017-03-05

RITAN is an R package intended for Rapid Integration of Term Annotation and Network resources. RITANdata is an R package containing data to support RITAN.

library(RITANdata)

Pathways, Terms, and Genesets

Network Resources

The R object, network_list, contains 6 human network-biology resources. Additionally, RITAN leverages existing R packages to access data from HPRD, BioGRID, and STRING. Citation information for each resource can be accessed via attr():

require(knitr)
## Loading required package: knitr
## Warning: package 'knitr' was built under R version 3.3.2
kable( attr(network_list, 'network_data_sources') )
